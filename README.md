  A simple Library Management System that depends on OOP concepts in Typescript.
  Project Ideas:
  ** In this projects users are allowed to:
      1- View books as cards.
      2- Search and filters books by book's name & author's name.
      3- Add new books (Regular & Reference).
      4- Toggle book availability.
      5- Remove books.
      6- Filter books by category.
  OOP Concepts Used:
    1- Encapsulation : All properties are private (#) and accessesd by (get,set).
    2- Inheritance : Class ReferenceBook extends class Book.
    3- Abstraction : Datas is only accessed through custom methods.
    4- Polymorphism : The function displayInfo() is overridden in the ReferenceBook Class.
  Technologies Used:
    * Html.
    * Css.
    * TypeScript.
